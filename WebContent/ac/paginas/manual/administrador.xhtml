Manual do administrador

Olá usuário(a) do módulo administrador, no sistema você possui permissões de gerenciar todos os usuários, cursos, turmas, grupos e atividades, abaixo será explicado o funcionamento do software para você.

Primeiramente a página de cadastro de administrador, na mesma você poderá cadastrar, alterar e excluir os mesmos.

Como pode ser observado, existe um botão cadastrar, acima da tabela que contém todos os administradores cadastrados, o mesmo botão abre um dialog onde será possível inserir os dados necessários. Na tabela é apresentados os botões de alterar e excluir, o primeiro irá abrir o mesmo dialog mencionado com as informações do registro selecionado, por fim o botão excluir que irá fazer com que o registro daquela linha seja removido, sumindo assim da tabela.

Para todos os outros cadastros seguirá a mesmas regras aplicadas acima, mas será exibido o cadastro de cadastro de professor e aluno, para que possa ser esclarecidas eventuais dúvidas, vale ressaltar nesse momento, se caso não seja inserida senha para os usuários o padrão para os mesmos será 123.

Na figura abaixo é exibido o cadastro de professor e é possível verificar que é permitido cadastrar mais de um curso para professor, sendo assim basta seleciona-los.


Para remover os cursos selecionados para professor deve seguir os seguintes passos clicar na lupa presente do lado direito da tabela nesse momento será aberto outro dialog apresentado abaixo, que contém um botão remover.


Na tela de cadastro de aluno, estão divididos entre ativos e inativos, para que possa ir para tabela de alunos trancados basta clicar sobre o botão de próximo, um ponto importante no cadastro de aluno, caso o mesmo seja alterado e mudado a sua turma as horas dos cadastros dos certificados do mesmo poderão ser alteradas, devido das horas computadas serem conforme a equivalência da turma.
